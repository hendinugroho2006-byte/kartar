# Karang Taruna (Static)
Buka `index.html` di browser.

## Cara ganti teks & data
- Data umum (nama, alamat, sosmed) ada di `js/data.js` (bagian `window.SITE`).
- Daftar kegiatan ada di `js/data.js` (bagian `window.KEGIATAN`).
- Foto galeri ada di `js/data.js` (bagian `window.GALERI`).
- Pengurus & divisi ada di `js/data.js` (bagian `window.PENGURUS` dan `window.DIVISI`).

## Cara ganti foto
Taruh file gambar di folder `assets/img/...` lalu ubah path-nya di `js/data.js`.
Contoh:
`image: "assets/img/kegiatan/lomba.png"`
